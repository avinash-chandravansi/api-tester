Quick guide

1. Enter the URL which you need to target. For Ex- http://localhost:3000

2. If you want to add certain payload , body or params to the request , click on Set Prameters button

3. After setting all the parameters , just click on make request button to genrated response output

>> In case of any difficulty , Just hit reset button to make fresh request <<
